FlowMVI: Build Resilient Multiplatform Apps

FlowMVI is a Kotlin Multiplatform architectural framework based on coroutines. It enables you to extend your business logic with reusable plugins, handle errors, achieve thread-safety, and more. It takes about 10 minutes to get started.

Get Started with FlowMVI

Why Choose FlowMVI?

Powerful Plug-In system

Automate processes and reuse business logic.

Automatic Error Recovery

Recover from errors and report them.

Async & Reactive

Build fully async, reactive apps with no manual thread sync.

Multiplatform Logic

Create multiplatform business logic with pluggable UI.

Lifecycle Handling

Automatic multiplatform system lifecycle handling.

Debugging & Logging

Out-of-the-box support for debugging, logging, caching.

State Management

Create compile-time safe state machines with a readable DSL.

Compose Multiplatform

First-class support, optimized for performance.

No Dependencies

Core library has no dependencies except Kotlin coroutines.
